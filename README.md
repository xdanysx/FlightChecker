# FlightChecker

Windows GUI-App: Roundtrip-Vergleich (Ryanair cheapestPerDay)

Diese App sucht die günstigste Zeitspanne zum fliegen von CGN nach PMO bzw. von NRN nach TPS. Sie nutzt dafür eine Ryanair API. Ergo werden nur nach Ryanair Flüge gescannt. Anschließend werden diese nochmal mithilfe eines Diagramms veranschaulicht gemacht.
